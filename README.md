# Capstone Automation Framework 
A Hybrid Test Automation Framework with TestNG, Cucumber (BDD), Page Object Model (POM), Extent Reporting, Logging, Listeners, and Excel Data Reader to automate a sample web application.

## Prerequisites

* **Java 23 (Open JDK)**
* **Maven**
* **Git**

## Running project on Local :
1. Clone the project `git clone <$RepositoryURL>`
2. Install dependencies `mvn install`
3. Run capstonetestng.xml as TestNG Suite.  
   <br />

## Reporting and Logging :
1. HTML Report will be generated under *HTMLExtentReport* folder.
2. Console logs will be generated under *logs* folder.
3. Screenshots will be generated under *ScreenshotResult* folder.
